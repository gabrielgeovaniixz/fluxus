# Fluxus - Dashboard Financeiro

O **Fluxus** é um sistema de controle financeiro pessoal desenvolvido para ajudar no gerenciamento de entradas e saídas de forma simples e intuitiva. Este projeto foi criado com o objetivo de colocar em prática conceitos modernos de React, como Hooks, CSS Modules e integração com bibliotecas de gráficos.

## 🚀 Funcionalidades

- **Dashboard Geral**: Visualize rapidamente seu saldo total, total de entradas e total de saídas.
- **Gráfico de Distribuição**: Acompanhe a proporção entre suas receitas e despesas através de um gráfico de pizza interativo.
- **Cadastro de Transações**: Adicione novas movimentações informando descrição, valor, categoria e data.
- **Histórico Completo**: Liste todas as suas transações e remova itens quando necessário.
- **Persistência de Dados**: Seus dados ficam salvos no navegador (LocalStorage), então você não perde as informações ao recarregar a página.
- **Design Responsivo**: Utilize o sistema tanto no computador quanto no celular.

## 🛠️ Tecnologias Utilizadas

- **React**: Biblioteca principal para construção da interface.
- **Vite**: Ferramenta de build extremamente rápida para o desenvolvimento.
- **CSS Modules**: Estilização isolada por componente para um código mais limpo e organizado.
- **Lucide React**: Biblioteca de ícones modernos e leves.
- **Recharts**: Para a criação dos gráficos financeiros.

## 📦 Como instalar e rodar

1. **Clone o repositório** (ou baixe os arquivos):
   ```bash
   git clone https://github.com/seu-usuario/fluxus.git
   ```

2. **Entre na pasta do projeto**:
   ```bash
   cd fluxus
   ```

3. **Instale as dependências**:
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```

5. Abra o navegador no endereço indicado (geralmente `http://localhost:5173`).
