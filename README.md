# Fluxus

Fluxus é um dashboard financeiro simples desenvolvido para controle de entradas e saídas financeiras.

O projeto foi criado utilizando React com Vite, focando em uma interface limpa, organizada e fácil de utilizar.

## Funcionalidades

- Cadastro de transações financeiras
- Controle de entradas e saídas
- Cálculo automático do saldo total
- Histórico de transações
- Remoção de transações
- Gráfico com resumo financeiro
- Salvamento automático no navegador com LocalStorage
- Layout responsivo

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS
- Recharts / Chart.js
- LocalStorage

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/gabrielgeovaniixz/fluxus.git
```

Acesse a pasta do projeto:

```bash
cd fluxus
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Depois disso, abra no navegador:

```bash
http://localhost:5173
```
